
# TV FREE - লাইভ টিভি অ্যাপ

**TV FREE** একটি অ্যান্ড্রয়েড অ্যাপ্লিকেশন যার মাধ্যমে ব্যবহারকারীরা বিনামূল্যে বিভিন্ন লাইভ টিভি চ্যানেল উপভোগ করতে পারবেন। এই অ্যাপটি ফায়ারবেস (Firebase) থেকে রিয়েল-টাইমে চ্যানেল ডেটা ফেচ করে এবং ExoPlayer-এর মাধ্যমে m3u8 লাইভ স্ট্রিম প্লে করে।

## 🚀 ফিচারসমূহ

* **লাইভ টিভি স্ট্রিমিং:** ExoPlayer ব্যবহার করে উচ্চ গতির m3u8 স্ট্রিম প্লেব্যাক।
* **রিয়েল-টাইম ডেটাবেস:** Firebase Firestore থেকে চ্যানেলের তালিকা স্বয়ংক্রিয়ভাবে আপডেট হয়।
* **সহজ চ্যানেল নেভিগেশন:** RecyclerView এবং GridLayoutManager ব্যবহার করে চ্যানেলগুলো সুন্দরভাবে সাজানো হয়েছে।
* **সার্চ অপশন:** সহজেই কাঙ্ক্ষিত চ্যানেল খুঁজে বের করার জন্য লাইভ সার্চ বার।
* **ফুলস্ক্রিন প্লেব্যাক:** ভিডিও দেখার অভিজ্ঞতা ভালো করতে ফুলস্ক্রিন মোড সাপোর্ট।
* **সাপোর্ট সিস্টেম:** ব্যবহারকারীরা যেকোনো সমস্যায় সরাসরি ডেভেলপারের সাথে যোগাযোগ করতে পারবেন।
* **লোগো লোডিং:** Coil লাইব্রেরি ব্যবহার করে ইন্টারনেট থেকে দ্রুত চ্যানেলের লোগো লোড করা হয়।

## 🛠️ ব্যবহৃত প্রযুক্তি

এই প্রোজেক্টটি তৈরি করতে নিম্নলিখিত প্রযুক্তি এবং লাইব্রেরিগুলো ব্যবহার করা হয়েছে:

* **ভাষা:** Kotlin
* **ইউজার ইন্টারফেস:** XML Layouts (অ্যান্ড্রয়েড ভিউ)
* **মিডিয়া প্লেয়ার:** Media3 ExoPlayer (HLS সাপোর্ট সহ)
* **ব্যাকএন্ড ও ডেটাবেস:** Firebase Firestore & Firebase Realtime Database
* **ইমেজ লোডিং:** Coil
* **কম্পোনেন্ট:** RecyclerView, ConstraintLayout

## ⚙️ কীভাবে প্রজেক্টটি সেটআপ করবেন

1.  প্রথমে গিটহাব থেকে রিপোজিটরিটি ক্লোন করুন:
    ```bash
    git clone https://github.com/biplobc384-dotcom/tvfree.git
    ```
2.  **Android Studio**-তে প্রজেক্টটি ওপেন করুন।
3.  আপনার নিজের একটি Firebase প্রজেক্ট তৈরি করুন এবং `google-services.json` ফাইলটি ডাউনলোড করে প্রজেক্টের `app/` ফোল্ডারের ভেতর রিপ্লেস করুন।
4.  আপনার Firebase Firestore-এ `channels` নামের একটি কালেকশন (Collection) তৈরি করুন। সেখানে প্রতিটি চ্যানেলের জন্য ডকুমেন্ট তৈরি করে নিচের ফিল্ডগুলো যুক্ত করুন:
    * `name` (String): চ্যানেলের নাম
    * `image` (String): চ্যানেলের লোগো বা ছবির URL
    * `url` (String): লাইভ স্ট্রিমের m3u8 লিংক
5.  প্রজেক্টটি **Sync** করুন এবং আপনার ডিভাইস বা এমুলেটরে **Run** করুন।
